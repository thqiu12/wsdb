# SchoolCore 帳票テンプレート監査 v1

更新日: 2026-05-10

## 目的

SchoolCore で自動生成される帳票が、既存の学校運用テンプレートに対してどこまで一致しているかを確認する。

今回の確認観点:

- テンプレートと同じシート構成か
- シート名が一致しているか
- 行列サイズが一致しているか
- 結合セル数が一致しているか
- 重要セルに値が入っているか

## 判定ルール

- `構造一致`: シート数、シート名、行列サイズ、結合セル数が実質一致
- `要目視確認`: 構造一致だが、見た目や余白、改ページ、印刷範囲は別途確認推奨
- `要修正`: 構造や重要値に問題がある

## 現時点の結論

今回監査した主要帳票は、少なくとも以下の意味では「テンプレートにかなり忠実」と判断できる。

- 別物の簡易表ではない
- テンプレートのシート構成を保っている
- 重要欄に値が入っている
- 半期報告、更新リスト、更新五表、年度終了報告、離脱届はかなり実運用に近い

ただし、`完全に無確認でそのまま提出可` と言い切る前に、次の1段階を推奨する。

## 次の確認推奨

1. 目視監査
   - 印刷レイアウト
   - 改ページ
   - 余白
   - フォント崩れ
   - 罫線のズレ

2. 実データ監査
   - 学校が実際に提出した過去帳票と並べて比較
   - 1件ずつ欄名ごとに照合

3. 運用判定
   - `そのまま提出可`
   - `提出前に人が一度見る`
   - `まだ修正必要`
   を帳票ごとに決める

## 推奨優先順位

次に優先して人手確認すべき帳票:

1. 入管5割出席率不佳報告
2. 半期毎出席率報告（総表 / 明細）
3. 在留期間更新許可申請書（個人五表）
4. 5月/11月 受け入れ状況在留者リスト

この4本が通れば、「入管向け主要帳票はかなり現実に近い」と言いやすくなる。
